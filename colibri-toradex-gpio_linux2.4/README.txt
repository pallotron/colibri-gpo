this directory contains a kernel module for linux2.4 i wrote to
handle the GPIO pins of the Toradex Colibri embedded system:

http://www.toradex.com/e/colibri_pxa270.php

to use this embedded system with linux and to build my kernel module
you need to download this tarball and install the arm toolchain on
you workstation:

http://vollmann.net/colibri/colibri-bsp-2.2.tar.gz

to install a linux os on the embedded you need to install dhcpd, nfs and
tftp on you workstation. follow the instruction on the tarball.
